04 — Motion & Easing

Token Valor Contexto
--ease-out cubic-bezier(0.4, 0, 0.2, 1) Transição padrão (ease-out)
--duration-fast 150ms Micro-interactions (raro)
--duration-normal 240ms Transição padrão (hover, focus)
--duration-slow 300ms Animações UI mais lentas
--duration-deliberate 600ms Progress bar, animações longas
--transition 240ms cubic-bezier(...) Shorthand padrão

Regra principal: 240ms é o padrão. Use 300ms para animações mais lentas (collapse, expand). Nunca use timing maior que 600ms.

09 — Princípios-Chave

Brutalismo Editorial Refinado

  • Tipografia é arquitetura: Headlines bold (peso 800), body neutro (peso 400). A tipografia define o espaço.
  • Cor como argumento: 1 accent cromático único (verde), paleta máxima 2 cores + neutrals + feedback set.
  • Estrutura visível: Bordas, linhas, espaço negativo como linguagem, não separadores técnicos. Máximo 3 níveis de hierarquia por tela.
  • Zero decoração: Nada de gradientes, sombras suaves, ícones flat 2015, arredondamentos, ilustrações ornamentais.
  • Comunicação direta: Labels curtos, verbos de ação, zero jargão. Copy que assume usuário inteligente.

Basta — Aplicação Específica

Sério, direto, do lado do usuário. Alta densidade informacional com hierarquia clara. Confiança sem condescendência. Brutalismo Editorial em versão afirmativa. Nunca genérico.

Padrões de Uso de Ícones

  • Inline com texto: 16px, vertical-align: middle
  • Em botões: 16px, gap 6px do texto (.btn-sm)
  • Standalone em badges: 12px, margin-right 4px
  • Em headers: 20px, cor gray-600
  • Em identidade bancária: 32px, cores de marca do banco

Formato Recomendado

SVG inline (não imagem). Sem fill fixo — herdar cor do texto com currentColor quando possível. Para cores específicas (bank badges), usar fill direto.
